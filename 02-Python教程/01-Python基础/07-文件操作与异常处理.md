# 文件操作与异常处理

## 7.1 文件操作基础

Python使用 `open()` 函数操作文件，支持多种模式。

### 打开文件

```python
# 基本语法
file = open("文件名", "模式", encoding="编码")

# 示例
file = open("test.txt", "r", encoding="utf-8")
```

### 文件模式

| 模式 | 含义 |
|------|------|
| `r` | 只读（默认），文件必须存在 |
| `w` | 写入，文件存在则覆盖，不存在则创建 |
| `a` | 追加，在文件末尾写入 |
| `x` | 新建写入，文件已存在则报错 |
| `b` | 二进制模式（如 `rb`、`wb`） |
| `+` | 读写模式（如 `r+`、`w+`） |

### 读取文件

```python
# 读取全部内容
file = open("test.txt", "r", encoding="utf-8")
content = file.read()
print(content)
file.close()

# 逐行读取
file = open("test.txt", "r", encoding="utf-8")
for line in file:
    print(line.strip())  # strip() 去除换行符
file.close()

# 读取所有行到列表
file = open("test.txt", "r", encoding="utf-8")
lines = file.readlines()
file.close()
```

### 写入文件

```python
file = open("output.txt", "w", encoding="utf-8")
file.write("第一行\n")
file.write("第二行\n")
file.close()

# 追加写入
file = open("output.txt", "a", encoding="utf-8")
file.write("第三行（追加）\n")
file.close()
```

## 7.2 使用 with 语句

`with` 语句会自动关闭文件，无需手动调用 `close()`：

```python
# 读取
with open("test.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)
# 文件在此处自动关闭

# 写入
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("使用 with 写入\n")
```

这是操作文件的推荐方式，可以避免忘记关闭文件导致的问题。

## 7.3 文件和目录操作

使用 `os` 和 `shutil` 模块操作文件和目录：

```python
import os
import shutil

# 当前工作目录
print(os.getcwd())

# 创建目录
os.makedirs("dir1/dir2", exist_ok=True)

# 列出目录内容
print(os.listdir("."))

# 判断路径类型
print(os.path.isfile("test.txt"))    # 是否是文件
print(os.path.isdir("dir1"))         # 是否是目录
print(os.path.exists("test.txt"))    # 是否存在

# 路径拼接
path = os.path.join("dir1", "dir2", "file.txt")
print(path)  # dir1\dir2\file.txt

# 获取文件名和扩展名
filename, ext = os.path.splitext("test.txt")
print(filename, ext)  # test .txt

# 复制文件
shutil.copy("source.txt", "dest.txt")

# 移动/重命名
os.rename("old.txt", "new.txt")

# 删除文件
os.remove("file.txt")

# 删除目录
os.rmdir("empty_dir")           # 删除空目录
shutil.rmtree("nonempty_dir")   # 删除非空目录
```

## 7.4 异常处理基础

程序运行时的错误称为异常。Python使用 `try-except` 捕获并处理异常。

### 基本语法

```python
try:
    result = 10 / 0
except ZeroDivisionError:
    print("除数不能为0")
```

### 捕获多种异常

```python
try:
    num = int(input("请输入数字："))
    result = 100 / num
    print(result)
except ValueError:
    print("输入的不是有效数字")
except ZeroDivisionError:
    print("不能除以0")
except Exception as e:
    print(f"未知错误：{e}")
```

### else 和 finally

```python
try:
    file = open("test.txt", "r")
    content = file.read()
except FileNotFoundError:
    print("文件不存在")
else:
    # 没有异常时执行
    print("文件读取成功，内容长度为：", len(content))
finally:
    # 无论是否异常都执行
    file.close()
    print("文件已关闭")
```

## 7.5 常见异常类型

| 异常类型 | 说明 |
|---------|------|
| `ZeroDivisionError` | 除以零 |
| `ValueError` | 值错误（如 int("abc")） |
| `TypeError` | 类型错误（如 "1" + 2） |
| `IndexError` | 索引越界 |
| `KeyError` | 字典键不存在 |
| `FileNotFoundError` | 文件不存在 |
| `AttributeError` | 对象属性不存在 |
| `ImportError` | 导入模块失败 |
| `TimeoutError` | 操作超时 |

## 7.6 自定义异常

```python
class PasswordError(Exception):
    """密码错误异常"""
    pass

def verify_password(password):
    if len(password) < 8:
        raise PasswordError("密码长度不足8位")
    if not any(c.isdigit() for c in password):
        raise PasswordError("密码必须包含数字")
    return True

try:
    verify_password("abc")
except PasswordError as e:
    print(f"密码验证失败：{e}")
```

## 7.7 实际应用：配置文件解析器

结合文件操作和异常处理，实现一个简单的配置文件读取工具：

```python
def read_config(filename):
    """
    读取配置文件，格式为 key=value
    返回字典
    """
    config = {}
    try:
        with open(filename, "r", encoding="utf-8") as f:
            for line_num, line in enumerate(f, 1):
                line = line.strip()
                # 跳过空行和注释
                if not line or line.startswith("#"):
                    continue
                if "=" not in line:
                    print(f"警告：第{line_num}行格式错误，已跳过")
                    continue
                key, value = line.split("=", 1)
                config[key.strip()] = value.strip()
    except FileNotFoundError:
        print(f"错误：配置文件 {filename} 不存在")
        return {}
    except PermissionError:
        print(f"错误：没有权限读取文件 {filename}")
        return {}
    
    return config

# 使用示例
config = read_config("settings.ini")
host = config.get("host", "localhost")
port = int(config.get("port", 8080))
print(f"连接至 {host}:{port}")
```

def word_count(filename):
    word_freq = {}
    try:
        with open(filename, "r", encoding="utf-8") as f:
            for line in f:
                words = line.strip().split()
                for word in words:
                    word = word.strip(".,!?\"'()").lower()
                    if word:
                        word_freq[word] = word_freq.get(word, 0) + 1
    except FileNotFoundError:
        print(f"文件 {filename} 不存在")
        return {}
    return word_freq

def safe_int_input(prompt):

        try:
            return int(input(prompt))
        except ValueError:
            print("输入无效，请输入整数")

# 3. 文件备份工具
def backup_file(filename):
    import shutil
    from datetime import datetime
    try:
        timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
        backup_name = f"{filename}.{timestamp}.bak"
        shutil.copy(filename, backup_name)
        print(f"备份成功：{backup_name}")
    except FileNotFoundError:
        print(f"文件 {filename} 不存在")
    except PermissionError:
        print("权限不足")
