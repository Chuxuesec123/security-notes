# 流程控制

## 4.1 条件判断（if/elif/else）

条件判断用于根据不同的条件执行不同的代码分支。

### if 语句

```python
age = 18

if age >= 18:
    print("已成年")
```

### if-else 语句

```python
age = 16

if age >= 18:
    print("已成年")
else:
    print("未成年")
```

### if-elif-else 语句

```python
score = 85

if score >= 90:
    grade = "优秀"
elif score >= 80:
    grade = "良好"
elif score >= 60:
    grade = "及格"
else:
    grade = "不及格"

print(grade)  # 良好
```

### 条件表达式

条件表达式是if-else的简写形式，也称为三元运算符：

```python
age = 20
status = "成年" if age >= 18 else "未成年"
print(status)  # 成年
```

### 比较运算符

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `==` | 等于 | `a == b` |
| `!=` | 不等于 | `a != b` |
| `>` | 大于 | `a > b` |
| `<` | 小于 | `a < b` |
| `>=` | 大于等于 | `a >= b` |
| `<=` | 小于等于 | `a <= b` |

### 逻辑运算符

| 运算符 | 含义 | 说明 |
|--------|------|------|
| `and` | 与 | 两边都为True才为True |
| `or` | 或 | 至少一边为True则为True |
| `not` | 非 | 取反 |

```python
score = 85
attendance = 0.9

# and：两个条件都满足
if score >= 60 and attendance >= 0.8:
    print("通过考核")

# or：满足一个条件即可
if score < 60 or attendance < 0.8:
    print("需要补考")

# not：条件取反
if not False:
    print("执行此代码")
```

## 4.2 循环（for）

`for` 循环用于遍历序列（列表、字符串、元组等）。

### 遍历列表

```python
fruits = ["apple", "banana", "orange"]

for fruit in fruits:
    print(fruit)
```

### 遍历字符串

```python
for char in "Python":
    print(char)
```

### range() 函数

`range()` 生成一个整数序列，常与for循环配合使用：

```python
# range(stop) —— 0 到 stop-1
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

# range(start, stop) —— start 到 stop-1
for i in range(2, 6):
    print(i)  # 2, 3, 4, 5

# range(start, stop, step) —— 带步长
for i in range(0, 10, 2):
    print(i)  # 0, 2, 4, 6, 8
```

### 使用 enumerate() 获取索引

```python
fruits = ["apple", "banana", "orange"]

for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")

# 输出：
# 0: apple
# 1: banana
# 2: orange
```

## 4.3 循环（while）

`while` 循环在条件为True时持续执行。

### 基本用法

```python
count = 0

while count < 5:
    print(count)
    count += 1  # 务必更新计数器，否则会无限循环
```

### 无限循环

在某些场景下（如服务器监听），需要无限循环：

```python
while True:
    cmd = input("请输入命令（输入exit退出）：")
    if cmd == "exit":
        break  # 退出循环
    print(f"执行命令：{cmd}")
```

## 4.4 循环控制语句

### break —— 跳出整个循环

```python
for i in range(10):
    if i == 5:
        break
    print(i)  # 输出：0, 1, 2, 3, 4
```

### continue —— 跳过当前迭代

```python
for i in range(5):
    if i == 2:
        continue  # 跳过i=2
    print(i)  # 输出：0, 1, 3, 4
```

### else 子句

`for` 和 `while` 循环可以带 `else` 子句，在循环正常结束（未被break中断）时执行：

```python
# 查找素数
n = 7
for i in range(2, n):
    if n % i == 0:
        print(f"{n}不是素数")
        break
else:
    print(f"{n}是素数")

# 输出：7是素数
```

## 4.5 循环嵌套

循环内部可以嵌套另一个循环：

```python
# 打印九九乘法表
for i in range(1, 10):
    for j in range(1, i + 1):
        print(f"{j}x{i}={i * j}", end="\t")
    print()  # 换行
```

## 4.6 实际应用示例

### 简单的密码验证

```python
password = "123456"
attempts = 3

while attempts > 0:
    pwd = input("请输入密码：")
    if pwd == password:
        print("登录成功")
        break
    else:
        attempts -= 1
        if attempts > 0:
            print(f"密码错误，还剩{attempts}次机会")
        else:
            print("账户已锁定")
```

```python
# 1. 猜数字游戏
import random

target = random.randint(1, 100)
while True:
    guess = int(input("猜一个数字(1-100)："))
    if guess == target:
        print("猜对了！")
        break
    elif guess < target:
        print("小了")
    else:
        print("大了")

n = 5
for i in range(n):
    print(" " * (n - i - 1) + "*" * (2 * i + 1))

# 3. 判断闰年
year = int(input("请输入年份："))

    print(f"{year}是闰年")
else:
    print(f"{year}不是闰年")
```
