# 模块与包

## 2.1 模块的概念

模块是一个 `.py` 文件，包含Python代码（函数、类、变量等）。通过模块可以将代码组织成独立的文件，便于管理和复用。

### 导入模块

```python
# 方式一：导入整个模块
import math
print(math.sqrt(16))  # 4.0

# 方式二：导入特定函数
from math import sqrt
print(sqrt(16))  # 4.0

# 方式三：导入多个
from math import sqrt, pi
print(sqrt(16), pi)  # 4.0 3.141592653589793

# 方式四：导入所有（不推荐，可能引起命名冲突）
from math import *

# 方式五：起别名
import math as m
print(m.sqrt(16))

from math import sqrt as square_root
print(square_root(16))
```

## 2.2 创建自定义模块

创建文件 `utils.py`：

```python
# utils.py
def add(a, b):
    return a + b

def multiply(a, b):
    return a * b

PI = 3.14159
```

在另一个文件中使用：

```python
# main.py
import utils

print(utils.add(3, 5))      # 8
print(utils.multiply(4, 7)) # 28
print(utils.PI)             # 3.14159
```

## 2.3 \_\_name\_\_ 变量

`__name__` 是Python内置变量：

- 当模块被直接运行时，`__name__` 值为 `"__main__"`
- 当模块被导入时，`__name__` 值为模块文件名

```python
# utils.py
def add(a, b):
    return a + b

# 仅当直接运行时执行下列代码
if __name__ == "__main__":
    # 测试代码
    print(add(3, 5))  # 8
    print("模块测试通过")
```

这种模式允许模块既可被导入使用，也可独立运行测试。

## 2.4 包（Package）

包是包含多个模块的目录，必须包含 `__init__.py` 文件（可以为空）。

### 包结构

```
my_package/
    __init__.py
    math_ops.py
    string_ops.py
    file_ops/
        __init__.py
        reader.py
        writer.py
```

### 包的导入

```python
# 导入包中的模块
import my_package.math_ops
my_package.math_ops.add(1, 2)

# 从包中导入
from my_package import math_ops
math_ops.add(1, 2)

# 从子包中导入
from my_package.file_ops import reader
```

### \_\_init\_\_.py 的用途

```python
# my_package/__init__.py

# 1. 简化导入路径
from .math_ops import add, subtract
from .string_ops import reverse

# 2. 定义包级别的变量
__version__ = "1.0.0"

# 3. 控制 from package import *
__all__ = ["add", "subtract", "reverse"]
```

## 2.5 第三方库的安装

使用 `pip` 安装第三方库：

```powershell
# 安装库
pip install requests
pip install beautifulsoup4

# 安装指定版本
pip install requests==2.28.0

# 安装多个库
pip install requests beautifulsoup4 lxml

# 卸载库
pip uninstall requests

# 查看已安装的库列表
pip list

# 导出依赖
pip freeze > requirements.txt

# 从文件安装依赖
pip install -r requirements.txt
```

## 2.6 常用内置模块

### sys —— 系统相关

```python
import sys

# 命令行参数
print(sys.argv)  # ['script.py', 'arg1', 'arg2']

# Python版本
print(sys.version)  # 3.12.0 ...

# 退出程序
sys.exit(1)  # 以状态码1退出
```

### os —— 操作系统接口

```python
import os

# 环境变量
print(os.environ.get("PATH"))

# 执行系统命令
os.system("ping 127.0.0.1")

# 路径操作
path = os.path.join("dir", "sub", "file.txt")
```

### datetime —— 日期时间

```python
from datetime import datetime, timedelta

now = datetime.now()
print(now)                       # 2026-06-23 15:30:00
print(now.strftime("%Y-%m-%d"))  # 2026-06-23
print(now.strftime("%H:%M:%S"))  # 15:30:00

# 时间计算
future = now + timedelta(days=7)
print(future)

# 字符串转时间
dt = datetime.strptime("2026-06-23", "%Y-%m-%d")
```

### json —— JSON数据

```python
import json

# Python对象 → JSON字符串
data = {"name": "Alice", "age": 25, "scores": [85, 92]}
json_str = json.dumps(data, ensure_ascii=False, indent=2)
print(json_str)

# JSON字符串 → Python对象
parsed = json.loads(json_str)
print(parsed["name"])  # Alice

# 读写JSON文件
with open("data.json", "w", encoding="utf-8") as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

with open("data.json", "r", encoding="utf-8") as f:
    data = json.load(f)
```

### hashlib —— 哈希计算

```python
import hashlib

# MD5哈希
text = "hello"
md5_hash = hashlib.md5(text.encode()).hexdigest()
print(md5_hash)  # 5d41402abc4b2a76b9719d911017c592

# SHA256哈希
sha256_hash = hashlib.sha256(text.encode()).hexdigest()
print(sha256_hash)
```

### argparse —— 命令行参数解析

```python
import argparse

parser = argparse.ArgumentParser(description="端口扫描工具")
parser.add_argument("host", help="目标主机地址")
parser.add_argument("-p", "--port", type=int, default=80, help="目标端口")
parser.add_argument("--timeout", type=float, default=3.0, help="超时时间")

args = parser.parse_args()

# 使用
print(f"扫描主机：{args.host}，端口：{args.port}，超时：{args.timeout}")
```
