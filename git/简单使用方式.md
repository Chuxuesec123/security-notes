# 初始化等基础流程
## 初始化仓库

```
git init
```
## 添加文件到暂存区

```bash
git add .
```
## 设置主分支

```bash
git branch -M main
```
## 关联远程仓库

```bash
git remote add origin 仓库地址
```
## 查看远程仓库

```bash
git remote -v 
```

# 推送代码
## 首次推送

```bash
git push -u origin main
```
## 普通推送

```bash
git push
```

# 拉取代码
## 同步远程最新代码

```
git pull origin main
```
## 解决历史不一致（常用）

```bash
git pull origin main --rebase
```
## 强制允许不同历史合并

```bash
git pull origin main --rebase --allow-unrelated-histories
```
