# 数据结构

## 5.1 列表（list）

列表是Python中最常用的数据结构，用方括号 `[]` 表示，元素之间用逗号分隔。

### 创建列表

```python
# 空列表
empty_list = []

# 包含元素的列表
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]  # 列表可以包含不同类型

# 使用 list() 创建
chars = list("Python")  # ['P', 'y', 't', 'h', 'o', 'n']
```

### 访问与修改

```python
fruits = ["apple", "banana", "orange", "grape"]

print(fruits[0])      # apple
print(fruits[-1])     # grape（负数索引从末尾开始）
print(fruits[1:3])    # ['banana', 'orange']（切片）

fruits[1] = "mango"   # 修改元素
print(fruits)         # ['apple', 'mango', 'orange', 'grape']
```

### 常用方法

```python
fruits = ["apple", "banana"]

# 添加元素
fruits.append("orange")    # 末尾添加：['apple', 'banana', 'orange']
fruits.insert(1, "grape")  # 指定位置插入：['apple', 'grape', 'banana', 'orange']

# 删除元素
fruits.remove("banana")    # 删除指定元素
popped = fruits.pop()      # 删除并返回末尾元素
fruits.pop(0)              # 删除并返回指定位置元素
fruits.clear()             # 清空列表

# 查找与统计
fruits = ["apple", "banana", "apple", "orange"]
print(fruits.index("banana"))  # 1（第一个匹配的索引）
print(fruits.count("apple"))   # 2（出现次数）
print("banana" in fruits)      # True（是否包含）

# 排序与反转
numbers = [3, 1, 4, 1, 5]
numbers.sort()           # 升序：[1, 1, 3, 4, 5]
numbers.sort(reverse=True)  # 降序：[5, 4, 3, 1, 1]
numbers.reverse()        # 反转顺序

# 列表复制
fruits_copy = fruits.copy()       # 浅拷贝
fruits_copy2 = fruits[:]          # 另一种拷贝方式
```

### 列表推导式

一种简洁的创建列表的方式：

```python
# 生成1-10的平方
squares = [x ** 2 for x in range(1, 11)]
print(squares)  # [1, 4, 9, 16, 25, 36, 49, 64, 81, 100]

# 带条件的列表推导式
evens = [x for x in range(10) if x % 2 == 0]
print(evens)  # [0, 2, 4, 6, 8]

# 嵌套循环的列表推导式
pairs = [(x, y) for x in [1, 2] for y in [3, 4]]
print(pairs)  # [(1, 3), (1, 4), (2, 3), (2, 4)]
```

## 5.2 元组（tuple）

元组用圆括号 `()` 表示，与列表类似，但是**不可修改**（不可变）。

```python
# 创建元组
point = (3, 4)
empty = ()
single = (1,)  # 注意：单元素元组必须加逗号

# 省略括号
another = 1, 2, 3  # 也是元组

# 访问（与列表相同）
print(point[0])      # 3
print(point[-1])     # 4
print(point[:1])     # (3,)

# 元组不可修改
# point[0] = 10     # 错误！元组不可修改

# 元组解包
x, y = point
print(x, y)  # 3 4

a, b, *rest = [1, 2, 3, 4, 5]
print(a)     # 1
print(b)     # 2
print(rest)  # [3, 4, 5]
```

**元组的用途**：
- 函数返回多个值
- 作为字典的键（列表不可作为键）
- 存储不应被修改的数据

## 5.3 字典（dict）

字典存储键值对，用花括号 `{}` 表示。

### 创建字典

```python
# 直接创建
user = {
    "name": "Alice",
    "age": 25,
    "is_admin": False
}

# 使用 dict() 创建
user2 = dict(name="Bob", age=30)

# 空字典
empty = {}
```

### 访问与修改

```python
user = {"name": "Alice", "age": 25}

# 访问
print(user["name"])       # Alice（键不存在会抛出KeyError）
print(user.get("age"))    # 25（键不存在返回None）
print(user.get("email", "未设置"))  # 未设置（设置默认值）

# 修改与添加
user["age"] = 26          # 修改现有键
user["email"] = "alice@example.com"  # 添加新键值对

# 删除
del user["age"]            # 删除指定键
email = user.pop("email")  # 删除并返回值
user.clear()               # 清空字典
```

### 遍历字典

```python
user = {"name": "Alice", "age": 25, "email": "alice@example.com"}

# 遍历键
for key in user:
    print(key)

# 遍历值
for value in user.values():
    print(value)

# 遍历键值对
for key, value in user.items():
    print(f"{key}: {value}")
```

### 字典推导式

```python
# 将字符串列表映射为长度
words = ["apple", "banana", "orange"]
word_lengths = {word: len(word) for word in words}
print(word_lengths)  # {'apple': 5, 'banana': 6, 'orange': 6}

# 过滤字典
scores = {"Alice": 85, "Bob": 92, "Charlie": 78}
passed = {name: score for name, score in scores.items() if score >= 80}
print(passed)  # {'Alice': 85, 'Bob': 92}
```

## 5.4 集合（set）

集合是无序、不重复的元素集合，用花括号 `{}` 表示。

```python
# 创建集合
numbers = {1, 2, 3, 3, 2, 1}  # 重复元素自动去重
print(numbers)  # {1, 2, 3}

# 使用 set() 创建
chars = set("hello")  # {'h', 'e', 'l', 'o'}

# 空集合必须用 set()，{} 创建的是空字典
empty = set()
```

### 集合运算

```python
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

print(a | b)   # 并集：{1, 2, 3, 4, 5, 6}
print(a & b)   # 交集：{3, 4}
print(a - b)   # 差集：{1, 2}（在a中不在b中）
print(a ^ b)   # 对称差：{1, 2, 5, 6}（不同时在两者中的元素）
```

### 集合常用方法

```python
fruits = {"apple", "banana"}

fruits.add("orange")         # 添加元素
fruits.remove("banana")      # 删除元素（不存在抛异常）
fruits.discard("grape")      # 删除元素（不存在不抛异常）
popped = fruits.pop()        # 删除并返回一个随机元素
fruits.clear()               # 清空集合
```

## 5.5 数据结构的选择

| 数据结构 | 是否有序 | 是否可变 | 是否允许重复 | 适用场景 |
|---------|---------|---------|-------------|---------|
| 列表 list | 是 | 是 | 是 | 有序序列、需频繁修改 |
| 元组 tuple | 是 | 否 | 是 | 固定数据、函数多返回值 |
| 字典 dict | 3.7+有序 | 是 | 键唯一 | 键值映射、快速查找 |
| 集合 set | 否 | 是 | 否 | 去重、集合运算 |

## 5.6 本章练习

```python
# 1. 统计列表中各元素的出现次数
items = ["a", "b", "a", "c", "b", "a"]
counts = {}
for item in items:
    counts[item] = counts.get(item, 0) + 1
print(counts)  # {'a': 3, 'b': 2, 'c': 1}

# 2. 两个列表合并为字典
names = ["Alice", "Bob", "Charlie"]
scores = [85, 92, 78]
result = dict(zip(names, scores))
print(result)  # {'Alice': 85, 'Bob': 92, 'Charlie': 78}

# 3. 列表去重并保持顺序
nums = [3, 1, 2, 3, 1, 4, 2]
unique = list(dict.fromkeys(nums))  # 利用字典键的唯一性
print(unique)  # [3, 1, 2, 4]
```

## 5.7 本章小结

本章详细介绍了Python四种核心数据结构：列表、元组、字典和集合。列表是最常用的序列类型，元组提供不可变的序列，字典适合键值映射，集合适用于去重和集合运算。根据实际需求选择合适的数据结构是编写高效Python代码的关键。
