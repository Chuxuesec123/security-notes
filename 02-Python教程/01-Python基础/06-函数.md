# 函数

## 6.1 函数的定义与调用

函数是一段可重复使用的代码块，用于完成特定的任务。

### 定义函数

使用 `def` 关键字定义函数：

```python
def greet():
    """打印问候语"""
    print("Hello, World!")
```

### 调用函数

```python
greet()  # Hello, World!
```

### 完整的函数结构

```python
def 函数名(参数列表):
    """文档字符串（可选）"""
    函数体
    return 返回值（可选）
```

## 6.2 函数参数

### 位置参数

按顺序传递参数：

```python
def add(a, b):
    return a + b

print(add(3, 5))  # 8
```

### 默认参数

为参数指定默认值，调用时可省略：

```python
def greet(name, greeting="Hello"):
    print(f"{greeting}, {name}!")

greet("Alice")           # Hello, Alice!
greet("Bob", "Hi")       # Hi, Bob!
```

**重要**：默认参数必须是不可变类型，不要使用列表或字典作为默认参数。

### 关键字参数

通过参数名称传递，不依赖顺序：

```python
def introduce(name, age, city):
    print(f"{name}，{age}岁，来自{city}")

introduce(age=25, name="Alice", city="北京")
# Alice，25岁，来自北京
```

### 可变参数

#### *args —— 接收任意数量位置参数

```python
def sum_all(*numbers):
    """接收任意数量的参数"""
    total = 0
    for n in numbers:
        total += n
    return total

print(sum_all(1, 2, 3))       # 6
print(sum_all(1, 2, 3, 4, 5)) # 15
```

#### **kwargs —— 接收任意数量关键字参数

```python
def print_info(**info):
    """接收任意数量的关键字参数"""
    for key, value in info.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=25, city="北京")
# name: Alice
# age: 25
# city: 北京
```

### 参数传递顺序

定义函数时，参数顺序必须遵循：

```
位置参数 → 默认参数 → *args → **kwargs
```

```python
def func(a, b, c=10, *args, **kwargs):
    print(a, b, c, args, kwargs)

func(1, 2, 3, 4, 5, x=10, y=20)
# 1 2 3 (4, 5) {'x': 10, 'y': 20}
```

## 6.3 返回值

使用 `return` 返回结果。函数可以返回多个值（本质是元组）：

```python
def divide(a, b):
    quotient = a // b
    remainder = a % b
    return quotient, remainder

q, r = divide(10, 3)
print(q, r)  # 3 1

# 无 return 的函数返回 None
def nothing():
    pass

print(nothing())  # None
```

## 6.4 作用域

### 局部变量

函数内部定义的变量，仅在函数内有效：

```python
def my_func():
    x = 10  # 局部变量
    print(x)

my_func()  # 10
# print(x)  # 错误：x未定义
```

### 全局变量

在函数外部定义的变量，可在函数内读取，但修改需使用 `global`：

```python
count = 0  # 全局变量

def increment():
    global count  # 声明使用全局变量
    count += 1

increment()
print(count)  # 1
```

## 6.5 匿名函数（lambda）

`lambda` 用于创建简单的匿名函数：

```python
# 普通函数
def double(x):
    return x * 2

# 等价的 lambda
double = lambda x: x * 2

print(double(5))  # 10
```

lambda常用于需要函数作为参数的场景：

```python
# 与 sorted 配合使用
students = [
    {"name": "Alice", "score": 85},
    {"name": "Bob", "score": 92},
    {"name": "Charlie", "score": 78}
]

sorted_students = sorted(students, key=lambda s: s["score"], reverse=True)
print(sorted_students)
# [{'name': 'Bob', 'score': 92}, {'name': 'Alice', 'score': 85}, {'name': 'Charlie', 'score': 78}]

# 与 filter、map 配合使用
numbers = [1, 2, 3, 4, 5, 6]
even = list(filter(lambda x: x % 2 == 0, numbers))     # [2, 4, 6]
squared = list(map(lambda x: x ** 2, numbers))          # [1, 4, 9, 16, 25, 36]
```

## 6.6 高阶函数

高阶函数是接受函数作为参数或返回函数的函数：

```python
def apply_twice(func, value):
    """将函数应用两次"""
    return func(func(value))

result = apply_twice(lambda x: x * 2, 5)
print(result)  # 20（先5*2=10，再10*2=20）
```

### 装饰器（入门）

装饰器是用于扩展函数功能的高级技巧：

```python
def timer(func):
    """计算函数执行时间"""
    def wrapper(*args, **kwargs):
        import time
        start = time.time()
        result = func(*args, **kwargs)
        end = time.time()
        print(f"{func.__name__} 执行耗时：{end - start:.4f}秒")
        return result
    return wrapper

@timer
def slow_function():
    total = sum(range(1000000))
    return total

slow_function()  # 输出执行时间
```

## 6.7 文档字符串

为函数编写文档是一种良好实践：

```python
def calculate_bmi(weight, height):
    """
    计算BMI指数
    
    参数:
        weight (float): 体重（kg）
        height (float): 身高（m）
    
    返回:
        float: BMI值
    
    示例:
        >>> calculate_bmi(70, 1.75)
        22.86
    """
    return weight / (height ** 2)

# 查看文档
help(calculate_bmi)
```

## 6.8 本章练习

```python
# 1. 编写函数判断一个数是否为素数
def is_prime(n):
    if n < 2:
        return False
    for i in range(2, int(n ** 0.5) + 1):
        if n % i == 0:
            return False
    return True

# 2. 编写函数计算斐波那契数列
def fibonacci(n):
    a, b = 0, 1
    result = []
    for _ in range(n):
        result.append(a)
        a, b = b, a + b
    return result

# 3. 编写函数验证密码强度
def check_password(password):
    if len(password) < 8:
        return "弱"
    has_upper = any(c.isupper() for c in password)
    has_lower = any(c.islower() for c in password)
    has_digit = any(c.isdigit() for c in password)
    if has_upper and has_lower and has_digit:
        return "强"
    elif has_upper or has_lower:
        return "中"
    return "弱"
```
