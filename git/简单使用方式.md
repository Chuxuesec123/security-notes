# 初始化等基础流程

1. 初始化仓库

```
git init
```

2. 添加文件到暂存区

```bash
git add .
```

3. 设置主分支

```bash
git branch -M main
```

4. 关联远程仓库

```bash
git remote add origin 仓库地址
```

5. 查看远程仓库

```bash
git remote -v 
```

# 推送代码

1. 首次推送

```bash
git push -u origin main
```

2. 普通推送

```bash
git push
```

# 拉取代码

1. 同步远程最新代码

```
git pull origin main
```

2. 解决历史不一致（常用）

```bash
git pull origin main --rebase
```

3. 强制允许不同历史合并

```bash
git pull origin main --rebase --allow-unrelated-histories
```
