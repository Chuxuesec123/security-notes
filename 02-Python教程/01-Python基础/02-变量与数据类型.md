# 变量与数据类型

## 2.1 变量

变量是存储数据的容器。在Python中，变量不需要声明类型，直接赋值即可创建。

```python
name = "Alice"      # 字符串
age = 25            # 整数
height = 1.68       # 浮点数
is_student = True   # 布尔值
```

### 变量命名规则

- 只能包含字母、数字和下划线
- 不能以数字开头
- 区分大小写（`name` 和 `Name` 是两个不同的变量）
- 不能使用Python保留关键字

```python
# 合法命名
user_name = "Bob"
userName = "Bob"
_user_name = "Bob"
name1 = "Bob"

# 非法命名
# 1name = "Bob"   错误：不能以数字开头
# my-name = "Bob" 错误：不能包含连字符
# class = "Bob"   错误：class 是保留关键字
```

### 保留关键字

Python有以下保留关键字，不可用作变量名：

```
False, None, True, and, as, assert, async, await, break,
class, continue, def, del, elif, else, except, finally,
for, from, global, if, import, in, is, lambda, nonlocal,
not, or, pass, raise, return, try, while, with, yield
```

## 2.2 基本数据类型

Python内置了以下几种基本数据类型：

| 类型 | 名称 | 示例 |
|------|------|------|
| 整数 | int | `42`, `-10`, `0` |
| 浮点数 | float | `3.14`, `-0.5`, `1.0` |
| 字符串 | str | `"Hello"`, `'Python'` |
| 布尔值 | bool | `True`, `False` |
| 空值 | NoneType | `None` |

### 整数（int）

整数是不带小数部分的数值，支持任意大小的整数：

```python
a = 100
b = -50
c = 0
d = 10 ** 100  # 大整数，Python可以处理
```

### 浮点数（float）

浮点数是带小数部分的数值：

```python
pi = 3.14159
gravity = 9.8
small = 1.23e-4  # 科学计数法，等价于 0.000123
```

浮点数计算可能存在精度问题：

```python
print(0.1 + 0.2)  # 输出 0.30000000000000004
```

### 字符串（str）

字符串是用引号括起来的文本：

```python
# 单引号
s1 = 'Hello'

# 双引号
s2 = "World"

# 三引号，支持多行
s3 = """这是
多行
字符串"""
```

### 布尔值（bool）

布尔值只有两个取值：`True` 和 `False`，常用于条件判断：

```python
is_ok = True
is_done = False
```

布尔值也可以参与数值运算，`True` 相当于1，`False` 相当于0。

### 空值（None）

`None` 表示"没有值"或"空"，不同于0或空字符串：

```python
result = None
```

## 2.3 类型判断与转换

### 查看变量类型

使用 `type()` 函数查看变量的数据类型：

```python
name = "Alice"
age = 25

print(type(name))  # <class 'str'>
print(type(age))   # <class 'int'>
```

### 类型转换

Python提供了多个类型转换函数：

```python
# 转整数
int("42")       # 42
int(3.14)       # 3（直接舍去小数部分）

# 转浮点数
float("3.14")   # 3.14
float(5)        # 5.0

# 转字符串
str(42)         # "42"
str(3.14)       # "3.14"

# 转布尔值
bool(1)         # True
bool(0)         # False
bool("")        # False
bool("Hello")   # True
```

## 2.4 输入与输出

### 输出 —— print()

`print()` 函数用于向屏幕输出内容：

```python
print("Hello")           # 输出字符串
print(42)                # 输出数字
print("Age:", 25)        # 输出多个值，自动用空格分隔
print("A", "B", "C")     # 输出：A B C
print("Hello", end="")   # 不换行
```

### 输入 —— input()

`input()` 函数用于从键盘获取用户输入，返回值始终是字符串：

```python
name = input("请输入你的名字：")
print("你好，" + name)

# 输入数字时需要转换类型
age = int(input("请输入你的年龄："))
print("明年你", age + 1, "岁")
```
