# 面向对象编程

## 1.1 面向对象基础

面向对象编程（OOP）是一种编程范式，将数据和操作数据的方法封装在"对象"中。

### 类与对象

- **类**：对象的模板，定义了属性（数据）和方法（行为）
- **对象**：类的实例，是具体的个体

```python
# 定义一个类
class Dog:
    """狗的类"""
    
    def __init__(self, name, age):
        """构造函数，创建对象时自动调用"""
        self.name = name  # 属性
        self.age = age
    
    def bark(self):
        """方法"""
        print(f"{self.name} 汪汪叫！")
    
    def info(self):
        return f"{self.name}，{self.age}岁"

# 创建对象（实例化）
dog1 = Dog("旺财", 3)
dog2 = Dog("小白", 1)

# 调用方法
dog1.bark()  # 旺财 汪汪叫！
print(dog1.info())  # 旺财，3岁
```

## 1.3 self 参数

- `self` 指向实例本身，是方法的第一个参数
- 通过 `self` 访问实例的属性和其他方法
- 调用方法时无需传递 `self`，Python自动传入

## 1.4 实例属性 vs 类属性

```python
class Person:
    # 类属性（所有实例共享）
    species = "human"
    count = 0
    
    def __init__(self, name):
        # 实例属性（每个实例独有）
        self.name = name
        Person.count += 1

p1 = Person("Alice")
p2 = Person("Bob")

print(p1.name)       # Alice（实例属性）
print(p1.species)    # human（类属性）
print(Person.count)  # 2（类属性，统计实例总数）
```

## 1.5 实例方法、类方法与静态方法

```python
class MathUtils:
    pi = 3.14159
    
    def __init__(self, value):
        self.value = value
    
    # 实例方法（需要self）
    def square(self):
        return self.value ** 2
    
    # 类方法（需要cls，可访问类属性）
    @classmethod
    def circle_area(cls, radius):
        return cls.pi * radius ** 2
    
    # 静态方法（无需self/cls，独立于类）
    @staticmethod
    def add(a, b):
        return a + b

m = MathUtils(5)
print(m.square())                  # 25（实例方法）
print(MathUtils.circle_area(3))    # 28.27431（类方法）
print(MathUtils.add(10, 20))       # 30（静态方法）
```

## 1.6 封装

封装将数据和方法隐藏在类的内部，通过访问控制保护数据：

```python
class BankAccount:
    def __init__(self, owner, balance=0):
        self.owner = owner
        self.__balance = balance  # 私有属性（双下划线开头）
    
    def deposit(self, amount):
        """存款"""
        if amount > 0:
            self.__balance += amount
            return f"存款{amount}元成功"
        return "金额无效"
    
    def withdraw(self, amount):
        """取款"""
        if 0 < amount <= self.__balance:
            self.__balance -= amount
            return f"取款{amount}元成功"
        return "余额不足或金额无效"
    
    def get_balance(self):
        """获取余额（受控访问）"""
        return self.__balance

account = BankAccount("Alice", 1000)
account.deposit(500)
# print(account.__balance)  # 错误：无法直接访问私有属性
print(account.get_balance())  # 1500（通过公开方法访问）
```

## 1.7 继承

继承允许一个类从另一个类获取属性和方法，实现代码复用：

```python
# 基类（父类）
class Animal:
    def __init__(self, name):
        self.name = name
    
    def speak(self):
        raise NotImplementedError("子类必须实现speak方法")

# 派生类（子类）
class Cat(Animal):
    def speak(self):
        return f"{self.name}：喵喵~"

class Dog(Animal):
    def speak(self):
        return f"{self.name}：汪汪！"

# 多态：同一个方法名，不同实现
animals = [Cat("小花"), Dog("旺财")]
for animal in animals:
    print(animal.speak())
# 小花：喵喵~
# 旺财：汪汪！
```

### super() —— 调用父类方法

```python
class Vehicle:
    def __init__(self, brand, speed=0):
        self.brand = brand
        self.speed = speed
    
    def info(self):
        return f"{self.brand}，速度：{self.speed}"

class Car(Vehicle):
    def __init__(self, brand, speed=0, fuel_type="汽油"):
        # 调用父类构造函数
        super().__init__(brand, speed)
        self.fuel_type = fuel_type
    
    def info(self):
        # 扩展父类方法
        base_info = super().info()
        return f"{base_info}，燃料：{self.fuel_type}"

car = Car("丰田", 120, "混动")
print(car.info())  # 丰田，速度：120，燃料：混动
```

## 1.8 特殊方法（魔术方法）

Python类中有很多以双下划线开头和结尾的特殊方法：

```python
class Book:
    def __init__(self, title, author, pages):
        self.title = title
        self.author = author
        self.pages = pages
    
    def __str__(self):
        """字符串表示（面向用户）"""
        return f"{self.title} - {self.author}"
    
    def __repr__(self):
        """字符串表示（面向开发者）"""
        return f"Book('{self.title}', '{self.author}', {self.pages})"
    
    def __len__(self):
        """len() 函数支持"""
        return self.pages
    
    def __eq__(self, other):
        """== 比较"""
        return self.title == other.title and self.author == other.author
    
    def __lt__(self, other):
        """< 比较（支持排序）"""
        return self.pages < other.pages

book1 = Book("Python入门", "张三", 200)
book2 = Book("Python入门", "张三", 200)
book3 = Book("安全工具开发", "李四", 350)

print(str(book1))     # Python入门 - 张三
print(len(book1))     # 200
print(book1 == book2) # True
print(book1 < book3)  # True（200 < 350）
```

## 1.9 实际应用：日志记录器

```python
import datetime

class Logger:
    """简单的日志记录器"""
    
    LEVELS = {
        "DEBUG": 10,
        "INFO": 20,
        "WARNING": 30,
        "ERROR": 40,
        "CRITICAL": 50
    }
    
    def __init__(self, name, level="INFO"):
        self.name = name
        self.level = self.LEVELS.get(level, 20)
    
    def _log(self, level, message):
        if self.LEVELS.get(level, 0) >= self.level:
            timestamp = datetime.datetime.now().strftime("%Y-%m-%d %H:%M:%S")
            print(f"[{timestamp}] [{level}] [{self.name}] {message}")
    
    def debug(self, msg):
        self._log("DEBUG", msg)
    
    def info(self, msg):
        self._log("INFO", msg)
    
    def warning(self, msg):
        self._log("WARNING", msg)
    
    def error(self, msg):
        self._log("ERROR", msg)

# 使用示例
log = Logger("扫描器", "INFO")
log.info("扫描开始")
log.debug("处理数据包")  # 不会输出，级别低于INFO
log.warning("发现异常端口")
log.error("连接超时")
```
