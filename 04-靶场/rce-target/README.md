# RCE（远程代码执行）靶场系统

## 📋 简介

基于RCE漏洞知识构建的综合性PHP RCE靶场。包含命令注入、代码执行、绕过技术等多个模块，适合安全学习与渗透测试练习。

## 🎯 功能模块

| 级别 | 模块 | 说明 |
|------|------|------|
| ✅ | 命令注入基础 | 多种命令执行函数演示 |
| ✅ | 命令注入（DVWA风格） | IP ping功能，分低/中/高级 |
| ✅ | 代码执行漏洞 | eval/assert等PHP代码执行 |
| ✅ | 绕过过滤挑战 | 空格/关键词/分隔符绕过 |
| ✅ | 文件读写利用 | 写入WebShell、读取敏感文件 |
| ✅ | 无回显RCE | DNS外带/时间盲注模拟 |
| ✅ | 反序列化RCE | PHP反序列化漏洞演示 |
| ✅ | 命令速查 | 内置常用命令速查表 |

## 🚀 部署方式

### 方式一：PHP内置服务器（推荐）

```bash
cd 04-靶场/rce-target
php -S 0.0.0.0:8080
```

访问：http://localhost:8080

### 方式二：Apache/Nginx + PHP

将本目录部署到Web服务器根目录即可。

### 方式三：Docker（需自行构建）

```bash
docker build -t rce-target .
docker run -d -p 8080:80 rce-target
```

## 🧪 使用说明

1. 首页选择要练习的模块
2. 每个模块有独立的说明和提示
3. 高级模块有WAF模拟，需使用绕过技术
4. 可查看源码学习漏洞原理
