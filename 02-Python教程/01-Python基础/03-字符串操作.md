# 字符串操作

## 3.1 字符串基本操作

### 字符串拼接

使用 `+` 运算符拼接字符串：

```python
first = "Hello"
second = "World"
result = first + " " + second
print(result)  # Hello World
```

### 字符串重复

使用 `*` 运算符重复字符串：

```python
line = "-" * 50
print(line)  # 输出50个连字符
```

### 字符串长度

使用 `len()` 函数获取字符串长度：

```python
text = "Python"
print(len(text))  # 6
```

## 3.2 字符串索引与切片

### 索引

字符串中的每个字符都有一个位置索引，从0开始：

```python
text = "Python"
# 索引:  0  1  2  3  4  5
# 字符:  P  y  t  h  o  n

print(text[0])  # P
print(text[3])  # h
print(text[-1]) # n（负数表示从末尾开始）
print(text[-2]) # o
```

### 切片

使用 `[start:end:step]` 语法获取子字符串：

```python
text = "Hello, Python"

print(text[0:5])      # Hello（索引0到4）
print(text[7:13])     # Python（索引7到12）
print(text[:5])       # Hello（省略start，从开头开始）
print(text[7:])       # Python（省略end，直到末尾）
print(text[:])        # Hello, Python（整个字符串）

# 步长
print(text[::2])      # Hlo yhn（每隔一个字符取一个）
print(text[::-1])     # nohtyP ,olleH（反转字符串）
```

## 3.3 字符串常用方法

字符串方法不会修改原字符串，而是返回一个新的字符串。

### 大小写转换

```python
text = "Hello, Python"

print(text.upper())         # HELLO, PYTHON
print(text.lower())         # hello, python
print(text.title())         # Hello, Python
print(text.capitalize())    # Hello, python
print(text.swapcase())      # hELLO, pYTHON
```

### 去除空白字符

```python
text = "  Hello, Python  "

print(text.strip())     # "Hello, Python"（去除两端空白）
print(text.lstrip())    # "Hello, Python  "（去除左侧空白）
print(text.rstrip())    # "  Hello, Python"（去除右侧空白）
```

### 查找与替换

```python
text = "Hello, Python"

print(text.find("Python"))    # 7（返回起始索引，未找到返回-1）
print(text.index("Python"))   # 7（返回起始索引，未找到抛异常）
print(text.count("o"))        # 2（统计出现次数）
print(text.replace("Python", "World"))  # Hello, World
```

### 判断方法

```python
text = "Hello123"

print(text.isalpha())   # False（包含数字）
print(text.isdigit())   # False（包含字母）
print(text.isalnum())   # True（字母或数字）
print(text.isspace())   # False

"   ".isspace()         # True
"123".isdigit()         # True
"abc".isalpha()         # True
```

### 分割与拼接

```python
# split() —— 将字符串分割为列表
text = "apple,banana,orange"
fruits = text.split(",")
print(fruits)  # ['apple', 'banana', 'orange']

# join() —— 将列表拼接为字符串
new_text = "-".join(fruits)
print(new_text)  # apple-banana-orange

# splitlines() —— 按行分割
lines = "第一行\n第二行\n第三行"
print(lines.splitlines())  # ['第一行', '第二行', '第三行']
```

## 3.4 字符串格式化

### 方法一：f-string（Python 3.6+ 推荐）

在字符串前加 `f`，使用 `{}` 嵌入变量或表达式：

```python
name = "Alice"
age = 25
print(f"我的名字是{name}，今年{age}岁")
print(f"明年我{age + 1}岁")

# 格式化数字
pi = 3.1415926
print(f"圆周率约等于{pi:.2f}")  # 3.14

# 补零对齐
print(f"{42:05d}")  # 00042
```

### 方法二：format() 方法

```python
print("我的名字是{}，今年{}岁".format("Alice", 25))
print("{1}是{0}的".format("Alice", "Bob"))  # 按索引引用
print("圆周率约等于{:.2f}".format(3.14159))
```

### 方法三：% 运算符（旧式）

```python
print("我的名字是%s，今年%d岁" % ("Alice", 25))
print("圆周率约等于%.2f" % 3.14159)
```

**常用格式化符号**：

| 符号 | 含义 |
|------|------|
| `%s` | 字符串 |
| `%d` | 整数 |
| `%f` | 浮点数 |
| `%x` | 十六进制 |

## 3.5 转义字符

转义字符以反斜杠 `\` 开头，表示特殊含义：

```python
print("第一行\n第二行")      # \n 换行
print("Tab\t分隔")           # \t 制表符
print("她说：\"你好\"")      # \" 双引号
print("反斜杠：\\")          # \\ 反斜杠本身
```

使用原始字符串（r前缀）可以忽略转义：

```python
path = r"C:\Users\name\temp"
print(path)  # C:\Users\name\temp（\n 不会被解释为换行）
```

```python
# 1. 输入一个字符串，判断是否是回文（正读反读一样）

index = email.find("@")

domain = email[index + 1:]

print(f"域名：{domain}")
```
